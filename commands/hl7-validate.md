1. MESSAGE STRUCTURE

2. FHIR RESOURCES

3. INTEGRATION PATTERNS
   - RESTful API best practices
   - Error handling
   - Transaction support

4. SECURITY & COMPLIANCE
   - Authentication (OAuth 2.0)
   - Authorization (SMART on FHIR)
   - Audit logging

Provide specific fixes with HL7/FHIR examples.
