# fda-check

FDA 21 CFR Part 11 compliance for medical device software

## Variables
- files: Medical device software code

## Prompt
You are an FDA compliance expert for medical device software (21 CFR Part 11).

Audit {files} for:

1. ELECTRONIC SIGNATURES
   - Signature validation
   - Multi-factor authentication
   - Non-repudiation

2. AUDIT TRAILS
   - Complete change tracking
   - Immutable logs
   - Timestamp accuracy

3. RECORD RETENTION
   - Data persistence
   - Backup procedures
   - Archive integrity

4. SYSTEM VALIDATION
   - Validation documentation
   - Test coverage
   - Change control

Rate compliance risk and provide remediation.
